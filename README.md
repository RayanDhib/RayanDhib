# Rayan Dhib

Computational scientist - Centre for mathematical Plasma Astrophysics, KU Leuven.
C++ · CUDA · MPI · Python

An index of my open-source work and related publications:

- **[COOLFluiD](https://github.com/andrealani/COOLFluiD)** - open-source multiphysics HPC platform. Developer of the high-order Flux Reconstruction solver, its implicit time integration (Jacobian-free Newton–Krylov, p-multigrid preconditioning), and its MHD extension. Related papers: [CPC 2024](https://doi.org/10.1016/j.cpc.2023.109006), [ApJ 2025](https://doi.org/10.3847/1538-4357/adace5), [ApJS 2026](https://doi.org/10.3847/1538-4365/ae4ec1).
- **[Qorona](https://github.com/RayanDhib/Qorona)** - synthetic coronal imagery from global MHD simulations; CPU and CUDA backends. Used in the CmPA–ESA prediction campaign for the August 2026 total solar eclipse. Paper submitted to ApJ.
- **[CFD-HOWL](https://github.com/RayanDhib/CFD-HOWL)** - I/O library for high-order CFD data ([SoftwareX 2024](https://doi.org/10.1016/j.softx.2024.101943)).
- **HelioFill** - latent diffusion model for solar farside EUV reconstruction, multi-GPU PyTorch ([arXiv:2510.17012](https://arxiv.org/abs/2510.17012)).
- **[Numba](https://github.com/numba/numba)** - merged compiler fix: [PR #10699](https://github.com/numba/numba/pull/10699).

Full publication list: [ORCID](https://orcid.org/0000-0003-3670-4678)
